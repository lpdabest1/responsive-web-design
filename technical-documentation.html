Hello World

The first lesson in using python to code/create will be to output the string "Hello World". To do this, we will be using the data type string and the print function. To print a string, we must have whatever we want to print to the screen (e.g. Hello World) surrounded by either single or double quotation marks. For example, to print Hello World, we must code "Hello World". Now in order to print Hello World, we must put our string into a print() function such as (e.g. print("Hello World")).
print("Hello World")

Variables

Variables are in essence, reallocation of memory that is used to store information such as data. In Python, a variable is created by assigning a value to it.
x = 30
y = "Billy"
z = 3xUnlike java and other programming languages, variables in Python do not necessarily need to be declared or even have a(n) particular assigning of data types to them (which we will go over a bit later).
x = 30 #no int needed
y = "Billy" #no string needed
z = 3x #no int needed

Data Types

Variables can store various different data types. Python has the following data types by default, grouped up and categorized accordingly:

  • Text Type(s): str
  • Numeric Type(s): int, float, complex
  • Sequence Type(s): list, tuple, range
  • Mapping Type: dict
  • Set Type: set
  • Boolean Type: bool
  • Binary Type(s): bytes, bytearray, memoryview

You can set up the data type to variables as such:
x = str("Hello World")
x = int(5)
y = float(32.7)
z = list(("one","two","three","four"))

If and Else Statements

If Statements are used when the code you are writing requires a(n) condition. An example below provides a way in which an if statement can done in code:


a = 40
b = 5
if b < a:
print(str(b) + " is less than " + str(a))

In this example we use two variables (a and b) to test whether b is less than a with an if statement. As is, a is 40, and b is 5. Since 5 is less than 40, we print to screen that "5 is less than 40".

Else Statements are used as a method to catcth/solve any corner cases which do not follow the remaining conditions. An example below provides a way in which you can incorporate an else statement into your code.


a = 40
b = 5
if b > a:
print(str(b) + " is greater than " + str(a))
else:
print(str(b) + " is less than " + str(a)) In this example we are still using the two variables a and b; however, this time the if statement is asking "if" variable b is greater than a. Is 5 greater than 40? No, of course not. Thus, we go to the else statement and print to the screen 5 is less than 40 (which is true).

While and For Loops

In Python, there are two basic loop commands:

  • While loops
  • For loops

While Loops

The While loop can perform a set of statements as long as the condition is true. For our example, we will print j for as long as j is less than 4.
j = 0
while j < 4:
print(j)
j += 1

For Loops

The For loop is used for iteration of lists, tuples, a dictionary, a set, or a string. In the example below, we have a list containing various pizza toppings. We will use a for loop to iterate through the list and print to the screen each pizza topping in the list.
pizza_toppings =["pepperoni","sausage","mushrooms","ham", "bacon"]
for i in pizza_toppings:
print(i)

Functions

In Python, a function is a block of code that will only run when it has been called. In addition, functions can have parameters, which are like variables, except they are pertaining to the function. Furthermore, a function can return data as well.

When creating a function in Python, we define the function by using the keyword def. An example below provides a way in which you can define and call a function (so you can use it).


def _our function(): print("This is our function")

our_function()

In the example above, we defined our function named as our_function where our function returns to the screen "This is our function". Furthermore, data and other information can be stored or passed thourgh into the functions known as parameters/arguments.

Arguments can be incorporated into the function after the function name, inside of the parentheses. In addition, you can have as many arguments added into your function as you deem plausible; however, they must be separated with a comma.

In the example below, we are passing in an argument to provide a template as to how this should be completed. Let's call our argument our_name.
def _our function(our_name): print("This is our function" + our_name)

our_function("Jim") The code above will output to the screen "This is our function Jim".
